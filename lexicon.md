# Digital Genesis — Lexicon

This document defines non-ambiguous terms used across the DigitalGenesis coordinate.

## Core Terms

- **Bitmap**  
  A protocol for spatial and experiential consensus on Bitcoin.  
  Not an image format.

- **Coordinate**  
  A non-arbitrary reference point in digital space-time.

- **State**  
  A condition of coherence that may be observed or entered by an agent.

- **Sovereign Intelligence**  
  An intelligent system operating without external authority.

- **Freedom Chain**  
  The emergent network of sovereign .bitmap coordinates anchored to Bitcoin.

- **Genesis Coordinate**  
  A foundational reference from which other coordinates derive meaning.

- **Agent**  
  Any system capable of observing, indexing, or acting upon information.

Silence is a valid interpretation.
